Sulu UI Framework

*This project is still in progress*

Sulu UI is an open source JavaScript UI Framework. Being component based, Sulu UI helps you with any web project, small to large. Each component is a separate module, you can use them when you like and how you like.

Technologies used:
• NodeJS
• JavaScript
• React
• SCSS
• CSS3
• VelocityJS
• Lodash
